Folder Arrange

Just unzip the main compressed file. And this is how you'll find all the files of this template. You'll just have to open these html files on your browser. You can start with index.html 

  1. css — folder with css files.
  2. fonts — folder with icon fonts.
  3. images — folder with all images.
  4. js — folder with Javascript files.
  5. js/vendor — folder with modern.js file.
  6. index.html — is for index Page.
  7. about.html — is for about Page.
  8. resume.html — is for resume Page.
  9. service.html — is for service Page.
  10. portfolio.html — is for portfolio Page.
  11. Blog.html — is for Blog Page.
  12. inner-blog.html — is for inner-blog Page.
  13. cantact.html — is for contact Page.
  14. process.php — is file for php.

FTP Upload

  • Open Your FTP manager and connect to your hosting
